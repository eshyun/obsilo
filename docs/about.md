# About
## Sebastian Hanke
Builder, product thinker, AI toolmaker.

I'm Sebastian, based in Germany. I built Obsilo because I wanted an AI that works inside my knowledge system rather than alongside it -- one that understands my notes, follows my workflows, and can actually do things.

## Why Obsilo
Working with Kilo Code in VS Code inspired what agentic loops should look like when operating inside another tool. Obsilo Agent is that idea applied to knowledge work.
